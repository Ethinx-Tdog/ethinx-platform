# Voice Assets

This directory contains voice/audio assets for content personalization.

## Expected Files

- `voice_intro.mp3` - Introduction audio clip (max 10 seconds)
- `voice_outro.mp3` - Closing audio clip (max 5 seconds)
- `voice_transition.mp3` - Transition sound effect (max 3 seconds)

## Adding Assets

Place your MP3 audio files in this directory with the exact filenames listed above.
The system will automatically detect and use these assets when generating content.
