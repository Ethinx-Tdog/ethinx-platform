# Image Assets

This directory contains avatar and image assets for content personalization.

## Expected Files

- `avatar_overlay.png` - Avatar overlay image for videos (200x200 pixels)
- `avatar_profile.png` - Profile avatar image (400x400 pixels)
- `avatar_thumbnail.png` - Small thumbnail avatar (50x50 pixels)

## Adding Assets

Place your PNG image files in this directory with the exact filenames listed above.
The system will automatically detect and use these assets when generating content.
